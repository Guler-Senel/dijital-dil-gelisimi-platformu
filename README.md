# dijital-dil-gelisimi-platformu
Okul öncesi dönemde Türkçe dil gelişimini merkeze alan, ekran süresi yönetimi ve görev tabanlı oyunlaştırmayı entegre eden modüler bir mobil platformdur. Yapay zeka destekli içerik önerileri ve ebeveyn denetimi ile çocukların öğrenme sürecini kişiselleştirir ve izler.
